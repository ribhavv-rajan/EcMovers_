EC Movers Ltd — Award Images
==============================

Place your award PNG files in this folder.

Recommended specs:
  - Format: PNG with transparent background
  - Size: 120×120 px (will display at 80×80 on page)
  - Naming: use descriptive names e.g. homestars-2024.png

To show a real image in the awards section:
  1. Put your image here, e.g.: assets/awards/award-2025.png
  2. Open index.html (and about.html)
  3. Find the .award-badge block you want to update
  4. Replace:
       <div class="award-icon">🏆</div>
     With:
       <img class="award-img" src="assets/awards/award-2025.png" alt="Best Mover 2025" />

For subpages (about.html) the path is the same: assets/awards/...
For location pages the path is: ../assets/awards/...
